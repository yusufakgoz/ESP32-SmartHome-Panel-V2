# 🏡 ESP32 Smart Home Panel V2

Bu proje, **ESP32 (CYD - Cheap Yellow Display)** kullanılarak geliştirilmiş, **LVGL** arayüz kütüphanesi ve **Arduino IoT Cloud** altyapısıyla çalışan profesyonel bir Akıllı Ev (Smart Home) Kontrol Panelidir. 

Önceki versiyonun (V1) üzerine donanımsal ve yazılımsal birçok iyileştirme yapılarak UI/UX standartları Apple iOS (Apple HomeKit) seviyesine çekilmiştir. Özel tasarım PLA 3D baskı kasa ile kullanıma sunulmuştur.

## ✨ Öne Çıkan Özellikler (V2)
* 📱 **Pürüzsüz Arayüz (LVGL):** Gecikmesiz, dokunmatik hassasiyeti yüksek, kaydırılabilir çoklu sayfa mimarisi.
* ☁️ **Bulut Entegrasyonu:** Arduino IoT Cloud ve Google Home entegrasyonu ile 0 gecikmeli çift yönlü haberleşme.
* 🌤️ **Canlı Hava Durumu & 5 Günlük Tahmin:** OpenWeatherMap API kullanılarak anlık veri çekimi ve Grid (Izgara) tasarımlı kayan yazılı tahmin ekranı.
* 🌓 **Aydınlık / Karanlık Tema (Light/Dark Mode):** Ayarlar menüsünden tek tuşla tüm arayüzün renk paletini anında değiştirme.
* 💡 **PWM Ekran Parlaklığı:** Slider (Kaydırıcı) üzerinden donanımsal arka ışık (backlight) kontrolü.
* 🔧 **Hardware Hack:** CYD ekranların fabrikasyon BGR renk sorunlarına (Invert Color) karşı yazılımsal düzeltme algoritmaları.

## 🛠️ Kullanılan Teknolojiler
* **Mikrodenetleyici:** ESP32 (2432S028 / CYD 2.8" Dokunmatik Ekran)
* **Dil:** C++ (Arduino IDE)
* **Kütüphaneler:** LVGL, TFT_eSPI, XPT2046_Touchscreen, ArduinoJson, ArduinoIoTCloud
* **Sensör:** DHT22 (İç ortam sıcaklık ve nem)
* **Dış Kasa:** PLA Filament ile 3D Baskı

## 🚀 Kurulum ve Kullanım
1. Bu depoyu indirin.
2. `arduino_secrets.h` dosyasının içerisine kendi Wi-Fi bilgilerinizi ve Arduino Cloud Secret Key'inizi girin.
3. Kodu ESP32 cihazınıza yükleyin.
4. Gerekli kütüphanelerin Arduino IDE üzerinde kurulu olduğundan emin olun (`lv_conf.h` ayarlarının ekranınıza uygun yapılması gerekmektedir).
